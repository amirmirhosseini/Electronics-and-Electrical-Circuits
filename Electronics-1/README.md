# Electronics I - Final Project (Three-Stage Transistor Amplifier)

This repository contains the simulation files and technical analysis for the **Electronics I** final project at the **University of Tehran** (Spring 2025).

## Project Overview & Architecture:
The project focuses on the theoretical design and **PSpice** simulation of a high-performance **Three-Stage Transistor Amplifier** using BJT transistors to optimize voltage gain, input impedance, and output swing.

### Circuit Stages:
* **Stage 1 (Common-Collector / Emitter-Follower):** Implemented for impedance matching, providing a high input resistance.
* **Stage 2 (Common-Emitter):** The core voltage amplification stage, designed to maximize the overall voltage gain ($A_v$).
* **Stage 3 (Common-Base):** Utilized to achieve high output impedance and excellent high-frequency response.

## Tools & Analyses
* **Software:** Cadence OrCAD / PSpice
* **Analyses Performed:** DC Operating Point (Q-Point), AC Sweep (Frequency Response), Input/Output Resistance calculations, and Maximum Output Voltage Swing evaluation.
