# Electronics II - Computer Assignment (Differential Amplifiers Simulation)

This repository contains the full simulation files and technical analysis for the **Electronics II** computer assignment at the **University of Tehran** (Fall 1404 / Semester 1404-1405).

##  Project Overview & Objectives
The project focuses on the theoretical design, optimization, and **NI Multisim** simulation of high-performance **Differential Amplifiers** to evaluate their performance under common-mode and differential-mode signals.

###  Core Simulation Phases:
* **Part 1 (BJT Differential Amplifier with Active Tail Current Source):** Designing and biasing a BJT-based differential pair to achieve a common-mode gain ($A_{cm}$) much smaller than 1, calculating the Q-point, and evaluating the Maximum Output Voltage Swing.
* **Part 2 (MOSFET Differential Amplifier with Cascade/Active Load):** Implementing a MOS-based differential pair, performing DC operating point analysis, and analyzing the impact of current variations on gain and swing.
* **Key Metrics Evaluated:** Differential-Mode Gain ($A_d$), Common-Mode Gain ($A_{cm}$), Common-Mode Rejection Ratio (**CMRR**), Output Resistance ($R_{out}$), and Mismatch Sensitivity.

##  Tools Used
* **Software:** NI Multisim (Used for all schematic captures and advanced AC/DC simulations)
